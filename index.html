I began my software development journey by learning about Html which serves as the foundational block of web architecture. HTML elements/tags include, but are not limited to, the following elements: h(heading 1,2,3,4,5,6), p(Paragraph), header, footer, div(container), section, nav, article (these three are mostly used to structure a blog), strong, em, span (these three are used to style text).

The beauty of the webpages lies in the styling, which is achieved using CSS. Styling can be done Inline (as an attribute to the elements), or through Internal and External methods. Internal involves using the style tag to target elements using the "class" and "id" attributes. External styling links a css file to the html file using the link tag, placed within the head of the Html document.
